# Movie Revue App
### Gives movie reviews of the valid movie searched
### Created with Vue.js, Bootstrap and TMDB API

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
